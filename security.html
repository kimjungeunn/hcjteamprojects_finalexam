DDOS 공격

웹

#좀비PC_집단공격 #서비스_마비

자세히 보기 ▾

DDoS(분산 서비스 거부) 공격은 여러 대의 컴퓨터를 동원해 특정 서버나 웹사이트에 비정상적으로 많은 트래픽을 한꺼번에 보내서, 시스템을 마비시키고 일반 사용자들이 이용하지 못하게 만드는 공격입니다. 쉽게 말해, 좁은 입구에 수많은 사람을 몰아넣어 아무도 출입하지 못하게 가로막는 것과 비슷합니다.

API injection

웹

#입력값_조작 #검증_누락 #데이터_유출

자세히 보기 ▾

공격자가 API 엔드포인트를 통해 악의적인 데이터를 주입하여 서버 측의 로직을 조작하거나, 데이터베이스를 탈취하고, 시스템 명령을 실행하는 보안 취약점을 말합니다.유형으론 SQL injection,NoSQL 인젝션,명령어 인젝션,등이 있다.

XSS

웹

#스크립트_삽입 #브라우저_실행 #쿠키_탈취

자세히 보기 ▾

악의적인 스크립트를 몰래 올려두어, 그 글을 클릭한 다른 사용자의 브라우저에서 해당 코드가 실행되게 만드는 공격입니다.이 공격을 통해 사용자의 로그인 세션 정보를 탈취하거나, 가짜 화면을 띄워 개인정보를 직접 입력하도록 유도하는 등 방문자를 직접적인 타깃으로 삼는 것이 특징입니다.

Broken Access Control

앱

#권한 #접근제어 #사용자

자세히 보기 ▾

액세스 제어가 취약하면 사용자는 주어진 권한을 벗어나 모든 데이터를 무단으로 열람∙수정 혹은 삭제할 수 있습니다. 대응 방안으로는 모든 리소스에 대한 접근 제어를 철저히 하고, 사용자에게 최소한의 권한만을 부여해야 합니다.

Insecure Design

#철저한 #효율 #설계

자세히 보기 ▾

안전하지 않게 설계된 애플리케이션은 개발 완료 후 코드를 수정하여도 보안 함을 완벽히 방어하는데 한계가 있습니다. 따라서 설계 단계에서부터 보안을 적절히 고려해야 합니다.

IP Header & ACL

네트워크

#출입명부 #네트워크입구 #안전제일

자세히 보기 ▾

네트워크 통신의 기본인 IP 주소를 확인하여 허락되지 않은 외부인의 접속을 입구에서부터 막는 기술입니다. IP 헤더 정보를 활용해 '누가 들어오고 나갈 수 있는지' 규칙을 세우는 네트워크 보안의 가장 기초적인 첫걸음입니다.

Secure Connection

네트워크

#가짜연결차단 #서버보호 #안전한통로

자세히 보기 ▾

서버에 가짜 연결 요청을 수없이 보내서 서버를 마비시키는 공격을 막아내는 기술입니다. TCP의 연결 과정을 응용하여 진짜 사용자만 서버에 접속할 수 있도록 걸러내어 중단 없는 서비스를 보장합니다.
